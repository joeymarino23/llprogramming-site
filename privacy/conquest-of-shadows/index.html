Privacy Policy – Conquest of Shadows

Effective Date: November 21, 2025

This privacy policy governs your use of the software application Conquest of Shadows ("Application") for mobile devices that was created by L&L Programming. The Application is a multiplayer strategy game where players build empires, train armies, explore territories, and engage in tactical combat on a persistent hex-based map.

Information We Collect

User Provided Information

The Application obtains the information you provide when you download and register the Application. Registration is required and uses Firebase Authentication.

  • Email address and password (via Firebase Authentication)
  • Player display name and character class selection
  • In-app purchase transaction information
  • Support communications

Gameplay Data

  • Game progress (levels, experience, dominion points)
  • Territory ownership and building placements
  • Army compositions and military actions
  • Dungeon exploration progress
  • Item inventory and spell selections
  • Resource generation and management data
  • Exploration mission data and rewards

Automatically Collected Information

  • Device type and operating system
  • Unique device identifiers
  • IP address
  • Usage analytics and session data

Cloud-Based Data Storage

Conquest of Shadows is an online multiplayer game that stores all game data using Google Firebase and Google Cloud Run to support a persistent multiplayer world.

Location Information

The Application does not collect precise real-time location data.

Children

We do not knowingly collect personal information from children under the age of 13.

Security

We use industry-standard safeguards including encryption, HTTPS communication, and server-side validation to protect your information.

Changes

This Privacy Policy may be updated from time to time. Continued use of the Application constitutes acceptance of any changes.
